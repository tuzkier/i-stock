# MyInvestment 看盘系统

一个本地运行的多市场股票看盘系统，覆盖美股、港股、A 股、韩股的自选、K 线、常用技术指标、综合买卖信号和提醒规则。

## 运行

```bash
npm install
npm run dev
```

后台行情只连接本机 FutuOpenD。启动应用前先确认 FutuOpenD 已运行，并监听 `127.0.0.1:11111`：

```bash
lsof -nP -iTCP:11111 -sTCP:LISTEN
```

打开 `http://localhost:4271`。

## 股票代码

系统界面仍接受常见代码输入，后台会转换为 FutuOpenD 代码：

- 美股：`AAPL`、`MSFT`、`TSLA`
- 港股：`0700.HK`、`9988.HK`，会转成 `HK.00700`、`HK.09988`
- A 股：`600519.SS`、`000001.SZ`，会转成 `SH.600519`、`SZ.000001`
- 韩股：`005930.KS`、`035420.KS`，会转成 `KR.005930`、`KR.035420`；能否返回行情取决于本机 FutuOpenD 权限和市场支持

当前行情代理只使用 FutuOpenD，不再内置 Yahoo 或演示 K 线。若 FutuOpenD 未启动、未登录、无权限或返回空数据，接口会返回 `unavailable` 来源状态，前端显示不可用提示，不伪造行情。

## 已实现功能

- 自选股本地保存
- 多市场股票添加和格式归一
- K 线、成交量、EMA20、EMA60、布林带
- RSI14、MACD、ATR14、成交量快照
- 买卖提醒：价格上穿/下破、强买点、强卖点
- 本地持久化：自选和提醒规则存入浏览器 `localStorage`

## 自研指标

技术信号的正式研究设计见 [技术信号研究设计](docs/technical-signal-research-design.md)。

当前代码里的“四因子共振指标”是早期草稿，后续会按研究设计升级为 `MTS` 多周期趋势评分模型。正式方向不是单指标判断，而是把主流技术分析方法组合成一个 `-100` 到 `100` 的评分：

- 趋势：价格与 EMA20、EMA20 与 EMA60、EMA20 斜率
- 动量：RSI、MACD 柱体方向和变化
- 波动率：布林带突破、布林带宽度、ATR 风险
- 成交量：当前量与 20 期均量、OBV 斜率

信号分层：

- `>= 65`：强信号
- `40 到 64`：确认
- `20 到 39`：观察
- `-19 到 19`：中性
- `-20 到 -39`：弱势观察
- `-40 到 -64`：风险确认
- `<= -65`：强风险

交易线：

- 观察买点：布林上轨或 EMA20
- 减仓线：布林中轨或 EMA20
- 风控止损：当前价减 `2.2 * ATR`，并参考布林下轨

这些信号只用于技术分析提醒，不构成投资建议。真实交易还需要结合基本面、流动性、仓位、交易成本和市场制度差异。
