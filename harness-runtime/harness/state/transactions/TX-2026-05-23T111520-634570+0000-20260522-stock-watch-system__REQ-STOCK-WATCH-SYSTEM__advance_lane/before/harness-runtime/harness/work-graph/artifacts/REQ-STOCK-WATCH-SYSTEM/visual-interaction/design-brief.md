# Visual Interaction Design Brief

## 目标

用一个可操作静态原型验证“自选工作台 + 看盘详情 + MTS 信号 + 提醒 + 来源降级”的单屏工作流。原型入口是 `prototype/index.html`，只呈现产品界面。

## 关键设计决策

- 采用桌面三栏布局：自选、图表详情、信号与提醒；移动端折叠为纵向卡片。
- 来源状态条全局常驻，避免用户误把降级行情当正式行情。
- MTS 使用解释卡，不使用单一买卖箭头。
- 提醒创建按“价格型 / 信号型”分流，并强制选择四级语义。
- 归档标的的提醒显示“因归档暂停”，保留恢复语义。
