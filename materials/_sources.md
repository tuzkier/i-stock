# 资料来源登记（provenance）

> 本文件登记 `materials/` 下每份外部资料的来源、获取方式与落盘位置。
> 规则：用户在对话里临时给的外部目录 / 链接 / 截图，**先落盘到 `materials/<子目录>/`、再在此登记**，
> 然后才能以 `materials/` 相对路径进入任务契约的 `source_materials`。未落盘 + 未登记的临时材料不进 `source_materials`。
